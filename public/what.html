SitePoint search

This was a weekend hobby project, developed as a part of a larger application for SitePoint's internal use.

Noticing how godawful WordPress' search is, I decided we needed something better and opted to use the newest (and somewhat experimental) technologies at my disposal:

  • Diffbot for crawling and searching
  • Diffbot PHP Client for API calls
  • Webiny's HTPL template engine for rendering
  • Bootstrap for UI

The full tutorial on how this was built is available on SitePoint.com, and you can see the source code here.
